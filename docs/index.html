Use cases

Built for CFO workflows

Real scenarios a finance leader can run in minutes, not hours.

Board Prep

Audit before you publish

Run a 7-check integrity scan and get a health score before locking board pack numbers. Broken links, formula errors and unit mismatches surface in seconds.

Monthly Close

Budget vs Actuals in 30 seconds

Compare two workbooks cell-by-cell. Filter to sheets that matter, set a variance threshold, and get a ranked table of favourable and unfavourable movements.

Planning

Bear / Base / Bull scenarios

Run named scenarios without touching the file. Override assumptions in natural language and see the impact on EBITDA and FCF instantly.

Investigation

"Why did this number change?"

Trace any cell's precedents up to N hops. Compare two model versions filtered to only the cells that feed a specific output — pinpoint the one changed assumption.

Data Lineage

Follow the data across files

Trace a revenue cell through external file links — from the operating model, through a data feed, back to the original CRM export.

Handover

Build a self-documenting model

Label key cells conversationally. Labels persist as a sidecar JSON file so any analyst can run get_all_labels and immediately understand the model's structure.

Installation

Up and running in 5 minutes

No cloud account needed. Everything runs locally on your machine.

1

Download the binary

Go to the latest release and download the binary for your platform.

⬇ macOS (Apple Silicon) ⬇ macOS (Intel)  ·  Windows
2

Move and make executable (macOS)

# Move to a permanent location mv ~/Downloads/forecast-mcp-mac-arm64 ~/Documents/ForecastAI/forecast-mcp chmod +x ~/Documents/ForecastAI/forecast-mcp # Create your Excel workspace folder mkdir -p ~/Documents/ForecastAI/models
3

Add to Claude Desktop config

Open ~/Library/Application Support/Claude/claude_desktop_config.json and add:

{ "mcpServers": { "forecast-ai-excel": { "command": "/Users/YOUR_NAME/Documents/ForecastAI/forecast-mcp", "env": { "EXCEL_WORKSPACE_DIR": "/Users/YOUR_NAME/Documents/ForecastAI/models", "MCP_TRANSPORT": "stdio" } } } }
4

Add your Excel files and restart Claude

Drop any .xlsx or .xls files into your models folder, then restart Claude Desktop. You'll see forecast-ai-excel in the MCP tools panel.

Try: "List my Excel files" to confirm it's working.

Developer install (Python)
If you prefer to run from source:
git clone https://github.com/me883/forecastAI cd forecastAI python3 -m venv .venv && source .venv/bin/activate pip install -e ".[dev]" forecast-mcp # verify it starts
